# SimpleOS

It is a simple operating system which reads from keyboard and displays the character on screen

USAGE:

make create - to build and run

make remove - to remove the built object and binary files

REQUIREMENTS:

1)gcc

2)nasm

3)ld(linker)

4)qemu
